If you like Mouseable, don't forget to hit the star button!

  • Ctrl+I to show suggestions. Ctrl+W to close window.
  • F1 to receive next key from OS.
  • F2 to save, F3 to save and apply.
  • F4 to create new config.
    • "A+B" for simultaneous input.
    "A - B" (Space required!) for sequential input.

    "Ctrl+C": Press Ctrl and press C before release Ctrl.
    "Ctrl - Ctrl": Press Ctrl, release it, and press again in key-timeout milliseconds.
    "A+B - C+D": Press A with B, release them, press C with D before timeout.

    Mouseable rely on strings from OS's API, so use F1 function well.
    Some keys may appear strange (ex: Num9,Num8) but ignore it. OS sent it.
